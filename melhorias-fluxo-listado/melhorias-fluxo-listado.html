    Três Leituras Honestas para Diretivos

    PERSPECTIVAS
    Os dados são os mesmos. A interpretação depende do horizonte e da prioridade do observador.
    📈
    Leitura 1 — A Positiva

    A melhoria de UX funcionou como deveria: convertemos melhor o mesmo público orgânico. Listado ativa acima da média total (+3–6pp MOB0) — o usuário já está no app MP.

    O custo de emissão maior é o preço do crescimento que se dilui com o empilhamento. Mar/26 gerou ~135K ativos via Listado — vs ~13K em Dez/25. Isso é crescimento real de base ativa.

    Mensagem: Investimento justificado. Focar em qualificar o crescimento, não revertê-lo.

    ⚠️
    Leitura 2 — A Cautelosa

    Emissão +15% acima do plano = custo real de R$9M no Q1. Abr/26: Listado Baixa Prop ativa 25% no MOB0 vs 49% nos outros canais com fricção — gap de −24pp. Payback Baixa Prop = 47 meses vs 30 meses Alta = +R$272 por cartão.

    Fev–Abr/26: ~518K emissões Baixa Prop incrementais via Listado. Sem fricção, esses usuários emitem sem a intenção real que o depósito R$50 sinalizaria. O gap de ativação cresce — não diminui.

    Mensagem: Crescimento com risco de qualidade. Controles de qualidade são urgentes — não opcionais.

    ♟️
    Leitura 3 — A Estratégica

    O novo baseline de 54–58% Listado não é reversível — é o novo normal. O plano precisa refletir isso. Manter o plano antigo cria alertas falsos toda semana.

    O argumento mais forte não é o custo de TD isolado: é o pipeline de TC. ~900K emissões orgânicas via app MP = base de upgrade para crédito. Esse é o ROI que justifica quando visto no P&L completo.

    Mensagem: Atualizar o plano, comunicar o novo cenário. O crescimento está certo — o enquadramento precisa evoluir.
